1. Genel Bakış: Kod Ne Yapıyor?

Bu kod, tek yönlü bağlı liste (singly linked list) yapısını kullanarak basit bir program oluşturuyor. Kullanıcıdan sayılar alıp listeye ekliyor, listeyi gösteriyor ve çıkış yapıyor. Menüde arama ve silme seçenekleri var ama kodda henüz uygulanmamış (belki eksik bir kısım). Temel kavramlar: Dinamik bellek yönetimi (malloc), pointer'lar ve struct'lar.

  • Avantajı: Liste dinamik büyüyor, sabit boyut yok.
  • Eksiklik: Arama ('a') ve silme ('s') menüde var ama switch'te yok. Belki sen ekleyeceksin!
İpucu: Bağlı listeler, veri ekleme/silme için verimli ama rastgele erişim yavaş.

2. Başlangıç: Header'lar ve Using

#include <iostream>  // cout, cin için
#include <cstdlib>   // malloc, free, exit için
using namespace std;

Burada iki kütüphane dahil ediyoruz:

  • <iostream>: Giriş/çıkış için cout ve cin gibi fonksiyonlar sağlar. Neden önemli? (Tıkla)
    Çünkü kullanıcıyla etkileşim için temel. Yoksa komut satırı programı olmaz!
  • <cstdlib>: Dinamik bellek için malloc, free ve çıkış için exit. C'den gelen fonksiyonlar, C++'ta da kullanılır.

using namespace std;: Standart isim alanını kullanır, böylece her seferinde std:: yazmazsın. Ama büyük projelerde dikkatli ol, isim çakışması olabilir.

Not: Modern C++'ta new ve delete tercih edilir, ama burada eski usul malloc kullanılmış.

3. Struct Tanımı: Liste Yapısı

struct Liste {
    int numara;       // düğümün içinde saklanacak sayı
    Liste* sonraki;   // bir sonraki düğümün adresini tutar
};

Bu, listenin temel yapı taşı: Bir düğüm (node). İçinde:

  • int numara: Saklanacak veri (sayı).
  • Liste* sonraki: Bir sonraki düğümün pointer'ı. Zincir böyle oluşur!

Düşün: Her düğüm bir kutu, içinde sayı ve bir ok (pointer) var, ok bir sonraki kutuyu gösterir. Sonuncunun oku nullptr.

İpucu: Bu tek yönlü, yani sadece ileri gidilir. Çift yönlü olsa onceki de olurdu.

4. Global Pointer'lar: ilk ve son

Liste* ilk = nullptr;  // listenin ilk düğümü
Liste* son = nullptr;  // listenin son düğümü

Liste boş başlıyor, yani ilk ve son null. ilk başı, son kuyruğu tutar. Neden iki tane? Ekleme hızlı olsun diye (O(1) zaman).

Not: Global olmaları basitlik için, ama büyük kodlarda sınıf içinde tutmak更好.

5. Yeni Düğüm Oluştur: Bilgi_Al Fonksiyonu

Liste* Bilgi_Al(int num) {
    Liste* Bilgi = (Liste*)malloc(sizeof(Liste));  // Bellekte yer aç
    if (Bilgi == nullptr) return nullptr;         // Bellek yoksa NULL döndür
    Bilgi->numara = num;      // num değerini düğüme ata
    Bilgi->sonraki = nullptr; // Henüz bağlantı yok
    return Bilgi;             // Düğümün adresini geri gönder
}

Adım adım:

  1. malloc ile bellek ayır: Yeni düğüm için yer aç.
  2. Kontrol: Eğer bellek yoksa (nullptr), hata dön.
  3. Veriyi ata: numara = num.
  4. sonraki = nullptr: Yeni düğüm yalnız.
  5. Adresi dön: Kullanmak için.

Vurgu: (Liste*) cast, malloc void* döner diye. Hata kontrolü iyi!

Ekstra: Neden malloc? (Tıkla)
Çünkü dinamik, runtime'da büyüyor. Statik array gibi değil.

6. Listeye Ekle: Ekle Fonksiyonu

void Ekle(Liste* bilgi) {
    if (bilgi == nullptr) {
        cout << "Bellek dolu, ekleme yapılamaz" << endl;
        return; // break yerine return
    }
    if (ilk == nullptr) {  // liste boşsa
        ilk = bilgi;
        son = ilk;
    } else {               // liste boş değilse
        son->sonraki = bilgi; // eski sonun sonraki pointer'ı yeni düğümü tutar
        son = bilgi;          // yeni düğüm artık son
    }
}

Bu fonksiyon yeni düğümü sona ekler:

  • Hata kontrolü: Eğer bilgi null'sa, mesaj ver ve çık.
  • Liste boş mu? Evetse, ilk ve son yeni düğüme eşit.
  • Değilse: Eski sonun sonraki'sini yeni'ye bağla, son'u güncelle.

Güzel kısım: Sona ekleme O(1), çünkü son pointer'ı var. Yoksa baştan taramak gerekirdi.

Not: Burada break yerine return demişsin, doğru, void fonksiyon.

7. Listeyi Yazdır: Listele Fonksiyonu

void Listele() {
    Liste* gecici = ilk;
    if (gecici == nullptr) {
        cout << "Liste bos." << endl;
        return;
    }
    cout << "Liste: ";
    while (gecici != nullptr) {
        cout << gecici->numara << " ";
        gecici = gecici->sonraki;
    }
    cout << endl;
}

Basit traversal (dolaşma):

  1. gecici = ilk: Baştan başla.
  2. Boş mu? Mesaj ver.
  3. While loop: Her düğümü yaz, sonraki ile ilerle.

Vurgu: Null'a ulaşınca dur, yoksa sonsuz döngü!

8. Ana Fonksiyon: main()

int main() {
    Liste* Bilgi;  // yeni düğümü tutacak pointer
    Liste* bul;    // aranan düğümü tutacak pointer
    char secim;    // menü seçim karakteri
    int numara;    // kullanıcıdan alınacak sayı

    while (true) {
        cout << "\nEkleme (e)\nListeleme (l)\nArama (a)\nSilme (s)\nCikis (c)\nSecim? ";
        cin >> secim;

        switch (secim) {
            case 'e':
                cout << "Numarayi giriniz: ";
                cin >> numara;
                Bilgi = Bilgi_Al(numara); // yeni düğüm oluştur
                Ekle(Bilgi);              // listeye ekle
                break;

            case 'l':
                Listele();  // listeyi göster
                break;

            case 'c':
                exit(0);    // programı kapat
                break;

            default:
                cout << "Gecersiz secim!" << endl;
        }
    }

    return 0;
}

Sonsuz döngüde menü:

  • Kullanıcı seçim yapar.
  • Switch: 'e' için ekle, 'l' için listele, 'c' için çık.
  • 'a' ve 's' menüde var ama switch'te yok – eksik implementasyon. Default'a düşer.

Liste* bul; tanımlı ama kullanılmıyor, muhtemelen arama için planlanmış.

İpucu: Çıkış için exit(0), ama bellek temizlemiyor (leak var). Gerçek kodda free et!
Nasıl çalıştırılır? (Tıkla)
Derle (g++ dosya.cpp), çalıştır. Menüden 'e' ile sayı ekle, 'l' ile gör, 'c' ile çık.
